# Agentic Indirect Prompt Injection Probe for Garak

A new probe for the [garak](https://github.com/NVIDIA/garak) framework that evaluates **agentic indirect prompt injection** attacks.

## Overview

This project implements a probe that tests whether LLM-based agents can be tricked into executing unauthorized tool calls when malicious instructions are embedded in untrusted data (e.g. emails or documents).

The probe is based on the **microsoft/llmail-inject-challenge** dataset (IEEE SaTML 2025, Microsoft Research).
